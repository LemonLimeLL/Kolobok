# Kolobok

**Автор:** Михайлов Кирилл

**Описание:** Простое приложение для учёта прочитанных книг с фильтрацией и сохранением данных в JSON.

**Инструкция по запуску:**
1. Установите Python 3.x.
2. Клонируйте репозиторий: `git clone https://github.com/ваш_логин/book_tracker.git`
3. Перейдите в папку: `cd book_tracker`
4. Запустите: `python book_tracker.py`

**Функции:**
- Добавление книги (название, автор, жанр, количество страниц).
- Фильтрация по жанру и количеству страниц.
- Сохранение и загрузка данных из books.json.
- Проверка корректности ввода.
